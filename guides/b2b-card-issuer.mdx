The Yativo Card Issuer Program lets businesses issue Visa/Mastercard virtual and physical cards directly to their own customers — under your brand. This guide covers the full setup: applying for issuer status, creating customers, issuing cards, and funding card balances programmatically.

<Note>
  The Issuer Program requires an invitation from Yativo. Contact your account manager or [support@yativo.com](mailto:support@yativo.com) to get started. The program section will not appear in your dashboard or API until your account has been enabled.
</Note>

## Who This Is For

- Fintechs and neobanks building branded card products
- Payroll and expense management platforms
- Crypto exchanges wanting to offer spending cards
- Any business that wants to issue cards to end users

<Steps>

<Step title="Apply for the issuer program">

Once your account manager has enabled the Card Issuer Program on your account, the section appears in your dashboard and the apply endpoint becomes available. Choose your funding network and optional IBAN add-on.

<CodeGroup>

```bash cURL
curl -X POST https://crypto-api.yativo.com/api/card-issuer/apply \
  -H "Authorization: Bearer eyJhbGciOiJSUzI1NiJ9.eyJzdWIiOiJ1c3JfMDFIWVo..." \
  -H "Content-Type: application/json" \
  -d '{
    "funding_structure": "master_wallet",
    "iban_enabled": false
  }'
```

```typescript TypeScript
import { YativoCrypto } from "@yativo/crypto-sdk";

const client = new YativoCrypto({
  apiKey: process.env.YATIVO_API_KEY!,
  baseUrl: "https://crypto-api.yativo.com/api/",
});

const application = await client.cardIssuer.apply({
  funding_structure: "master_wallet", // "master_wallet" = SOL | "non_master" = XDC
  iban_enabled: false,
});

console.log(application.status); // "pending"
```

```python Python
from yativo_crypto import YativoCrypto

client = YativoCrypto(
    api_key=os.environ["YATIVO_API_KEY"],
    base_url="https://crypto-api.yativo.com/api/",
)

application = client.card_issuer.apply(
    funding_structure="master_wallet",
    iban_enabled=False,
)

print(application.status)  # "pending"
```

</CodeGroup>

**Funding structure options:**

| Value | Network | Best for |
|---|---|---|
| `master_wallet` | Solana | High volume, ~2–5 s settlement |
| `non_master` | XDC Network | Enterprise, near-zero fees, ~2 s settlement |

</Step>

<CodeGroup>

  -H "Authorization: Bearer eyJhbGciOiJSUzI1NiJ9.eyJzdWIiOiJ1c3JfMDFIWVo..."
```

console.log(program.issuerId);  // iss_01J2KN3MNPQ7R4S5T6U7VISSUER
console.log(program.limits.maxActiveCards);    // 10000

<Step title="Create customers">

  -H "Authorization: Bearer eyJhbGciOiJSUzI1NiJ9.eyJzdWIiOiJ1c3JfMDFIWVo..." \
  -H "Content-Type: application/json" \
  -d '{
    "externalId": "your_user_id_8821",
    "firstName": "Priya",
    "lastName": "Sharma",
    "email": "priya@yourapp.com",
    "phoneNumber": "+919876543210",
    "dateOfBirth": "1992-11-28",
    "nationality": "IN",
    "address": {
      "line1": "42 MG Road",
      "city": "Bangalore",
      "state": "KA",
      "postalCode": "560001",
      "country": "IN"
    }

```

<Step title="Fund the customer's card">

Transfer funds from your master wallet balance to a customer's card. Identify the customer by **email**, **Yativo card ID** (`yc_...`), or your own **external reference** — no need to look up IDs first.

<CodeGroup>

```bash cURL — fund by email
curl -X POST "https://crypto-api.yativo.com/api/card-issuer/fund" \
  -H "Authorization: Bearer eyJhbGciOiJSUzI1NiJ9.eyJzdWIiOiJ1c3JfMDFIWVo..." \
  -H "Content-Type: application/json" \
  -d '{
    "identifier": "priya@yourapp.com",
    "amount": 250.00,
    "currency": "USD"
  }'
```

```bash cURL — fund by external reference
curl -X POST "https://crypto-api.yativo.com/api/card-issuer/fund" \
  -H "Authorization: Bearer eyJhbGciOiJSUzI1NiJ9.eyJzdWIiOiJ1c3JfMDFIWVo..." \
  -H "Content-Type: application/json" \
  -d '{
    "identifier": "your_user_id_8821",
    "amount": 100.00,
    "currency": "EUR"
  }'
```

```typescript TypeScript
// Fund by email
const funding = await client.cardIssuer.fundCustomer({
  identifier: "priya@yourapp.com",
  amount: 250.00,
  currency: "USD",
});

// Fund by your own user ID (external_reference set at onboarding)
const funding = await client.cardIssuer.fundCustomer({
  identifier: "your_user_id_8821",
  amount: 100.00,
  currency: "EUR",
});

console.log(funding.status); // "completed"
```

```python Python
# Fund by email

    identifier="priya@yourapp.com",
    amount=250.00,
    currency="USD",

print(funding.status)  # "completed"
```

</CodeGroup>

<Step title="Monitor with webhooks">

Subscribe to card issuer events to keep your platform in sync.

<CodeGroup>

```typescript TypeScript
const webhook = await client.webhooks.create({
  url: "https://api.yourapp.com/webhooks/yativo",
  events: [
    "card.transaction.approved",
    "card.transaction.declined",
    "card.funding.completed",
    "customer.kyc.approved",
    "customer.kyc.rejected",
  ],
  description: "Issuer program events",
});
```

</CodeGroup>

**Relevant events for card issuers:**

| Event | Trigger |
|-------|---------|
| `card.transaction.approved` | A card transaction was authorized |
| `card.transaction.declined` | A transaction was declined (includes reason) |
| `card.transaction.completed` | A settlement cleared |
| `card.funding.completed` | A top-up completed |
| `customer.kyc.approved` | A customer's KYC was approved |
| `customer.kyc.rejected` | A customer's KYC was rejected |
| `card_issuer.limit_warning` | Your program is approaching a volume limit |

## Bulk Card Issuance

For issuing cards to many customers at once, batch the create-customer and create-card calls with appropriate rate limiting:

```typescript TypeScript
async function bulkIssueCards(users: UserRecord[]): Promise<void> {
  const BATCH_SIZE = 10;
  const DELAY_MS = 200; // stay within rate limits

  for (let i = 0; i < users.length; i += BATCH_SIZE) {
    const batch = users.slice(i, i + BATCH_SIZE);

    await Promise.all(
      batch.map(async (user) => {
        const customer = await client.customers.create({
          externalId: user.id,
          firstName: user.firstName,
          lastName: user.lastName,
          email: user.email,
          // ...
        });

        const card = await client.card.createCardForCustomer({
          customerId: customer.customerId,
          type: "virtual",
          label: `${user.firstName} Card`,
          currency: "USD",
        });

        await db.users.update(user.id, {
          yativoCustomerId: customer.customerId,
          yativoCardId: card.cardId,
        });
      })
    );

    if (i + BATCH_SIZE < users.length) {
      await new Promise((r) => setTimeout(r, DELAY_MS));
    }
  }
}
```
