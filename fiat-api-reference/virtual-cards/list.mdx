---
title: "List Virtual Cards"
description: "Retrieve all virtual cards issued to customers"
---

Returns all virtual cards issued under your business account.

```
GET /customer/virtual/cards/list
```

## Query Parameters

<ParamField query="page" type="number">
  Page number.
</ParamField>

<ParamField query="per_page" type="number">
  Results per page.
</ParamField>

<RequestExample>
```bash cURL
curl -X GET 'https://api.yativo.com/api/v1/customer/virtual/cards/list?page=1&per_page=20' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```
</RequestExample>
