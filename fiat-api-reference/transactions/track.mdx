---
title: "Track Transaction"
description: "Get a live status timeline for a transaction"
---

Returns a step-by-step timeline of a transaction's progress from initiation through completion.

```
GET /transaction/tracking/{transactionId}
```

## Path Parameters

<ParamField path="transactionId" type="string" required>
  The transaction UUID (`transaction_id` field from the transaction object).
</ParamField>

<RequestExample>
```bash cURL
curl -X GET 'https://api.yativo.com/api/v1/transaction/tracking/a0e9e50e-81be-4bd0-9941-0151e68b9e97' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```
</RequestExample>
