---
title: "Get Occupation Codes"
description: "Retrieve the list of valid occupation codes for KYC submissions"
---

Returns all valid occupation codes to use in the `most_recent_occupation_code` field of individual KYC submissions.

```
GET /auth/occupation-codes
```

<RequestExample>
```bash cURL
curl -X GET 'https://api.yativo.com/api/v1/auth/occupation-codes' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```
</RequestExample>
