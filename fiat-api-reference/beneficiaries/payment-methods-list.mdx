---
title: "List Payment Methods"
description: "Retrieve all saved payment methods across all beneficiaries"
---

Returns all payment methods saved for all beneficiaries in your account.

```
GET /beneficiaries/payment-methods/all
```

<RequestExample>
```bash cURL
curl -X GET 'https://api.yativo.com/api/v1/beneficiaries/payment-methods/all' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```
</RequestExample>
