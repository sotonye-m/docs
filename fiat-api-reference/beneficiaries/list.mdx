---
title: "List Beneficiaries"
description: "Retrieve all saved beneficiaries in your account"
---

Returns a paginated list of all beneficiaries associated with your account.

```
GET /beneficiaries
```

<RequestExample>
```bash cURL
curl -X GET 'https://api.yativo.com/api/v1/beneficiaries' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```
</RequestExample>
