---
title: "Get Customer"
api: "GET https://crypto-sandbox.yativo.com/api/customers/{customerId}"
description: "Retrieve a single customer by ID"
authMethod: "bearer"
---

<ParamField header="Authorization" type="string" required>
  Bearer token: `Bearer YOUR_ACCESS_TOKEN`
</ParamField>

<ParamField path="customerId" type="string" required>
  The unique customer ID.
</ParamField>

<Accordion title="Response Type Definitions">
```typescript
interface CustomerDetail {
  id: string;
  email: string;
  first_name: string;
  last_name: string;
  phone?: string;
  status: "active" | "suspended" | "pending";
  kyc_status: "not_started" | "pending" | "approved" | "rejected";
  wallets?: Array<{ address: string; network: string; balance: string }>;
  created_at: string;
  updated_at: string;
}
```
</Accordion>

<RequestExample>
```bash cURL
curl -X GET 'https://crypto-api.yativo.com/api/customers/cust_01HX9KZMB3F7VNQP8R2WDGT4E5' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```
</RequestExample>
