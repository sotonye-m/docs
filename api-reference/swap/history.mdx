---
title: "Swap History"
api: "GET /v1/swap/history"
description: "Retrieve the history of swap transactions"
authMethod: "bearer"
---

<ParamField header="Authorization" type="string" required>
  Bearer token: `Bearer YOUR_ACCESS_TOKEN`
</ParamField>

<ParamField query="limit" type="integer">
  Maximum number of swaps to return. Defaults to `20`.
</ParamField>

<ParamField query="offset" type="integer">
  Number of records to skip for pagination. Defaults to `0`.
</ParamField>

<ParamField query="status" type="string">
  Filter by swap status. One of: `pending`, `completed`, `failed`.
</ParamField>

<RequestExample>
```bash
curl -X GET 'https://crypto-api.yativo.com/api/v1/swap/history?limit=20&offset=0&status=completed' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```
</RequestExample>

<ResponseExample>
```json
{"success": true, "data": []}
```
</ResponseExample>
