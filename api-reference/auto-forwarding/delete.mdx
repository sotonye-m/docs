---
title: "Delete Rule"
api: "DELETE /v1/auto-forwarding/{rule_id}"
description: "Delete an auto-forwarding rule"
authMethod: "bearer"
---

<ParamField header="Authorization" type="string" required>
  Bearer token: `Bearer YOUR_ACCESS_TOKEN`
</ParamField>

<ParamField path="rule_id" type="string" required>
  The rule ID to delete.
</ParamField>

<RequestExample>
```bash
curl -X DELETE '/v1/auto-forwarding/af_01abc123' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "message": "Forwarding rule deleted"
}
```
</ResponseExample>
