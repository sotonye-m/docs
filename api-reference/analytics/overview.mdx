---
title: "Analytics Overview"
api: "GET https://crypto-sandbox.yativo.com/api/analytics/overview"
description: "Get a high-level summary of transaction volume, active accounts, and platform activity"
authMethod: "bearer"
---

<ParamField header="Authorization" type="string" required>
  Bearer token: `Bearer YOUR_ACCESS_TOKEN`
</ParamField>

<ParamField query="from" type="string">
  Start date for the reporting period. ISO 8601 format: `2026-03-01`
</ParamField>

<ParamField query="to" type="string">
  End date for the reporting period. ISO 8601 format: `2026-03-31`
</ParamField>

<Accordion title="Response Type Definitions">
```typescript
interface AnalyticsOverviewResponse {
  status: "success";
  data: {
    period: { from: string; to: string };
    transactions: {
      total_count: number;
      total_volume_usd: string;
      successful: number;
      failed: number;
      pending: number;
    };
    accounts: {
      total: number;
      active: number;
      new_this_period: number;
    };
    top_assets: Array<{
      symbol: string;
      network: string;
      volume_usd: string;
      count: number;
    }>;
  };
}
```
</Accordion>

<RequestExample>
```bash cURL
curl -X GET 'https://crypto-api.yativo.com/api/analytics/overview?from=2026-03-01&to=2026-03-31' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```
</RequestExample>
