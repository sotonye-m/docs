---
title: "Volume Analytics"
api: "GET https://crypto-sandbox.yativo.com/api/analytics/dashboard-analytics"
description: "Get volume and chart analytics for your account."
authMethod: "bearer"
---

<ParamField header="Authorization" type="string" required>
  Bearer token: `Bearer YOUR_ACCESS_TOKEN`
</ParamField>

<ParamField query="from" type="string" required>
  Start date. ISO 8601 format: `2026-03-01`
</ParamField>

<ParamField query="to" type="string" required>
  End date. ISO 8601 format: `2026-03-31`
</ParamField>

<ParamField query="group_by" type="string">
  Dimension to group by. Accepted values: `asset`, `network`, `asset_network`. Default: `asset`
</ParamField>

<Accordion title="Response Type Definitions">
```typescript
interface VolumeAnalyticsResponse {
  status: "success";
  data: {
    period: { from: string; to: string };
    breakdown: Array<{
      asset?: string;
      network?: string;
      volume_usd: string;
      volume_native: string;
      count: number;
      percentage: number;
    }>;
    total_volume_usd: string;
  };
}
```
</Accordion>

<RequestExample>
```bash cURL
curl -X GET 'https://crypto-api.yativo.com/api/analytics/dashboard-analytics' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```
