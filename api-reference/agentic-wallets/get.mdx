---
title: "Get Agentic Wallet"
api: "GET /v1/agentic-wallets/{walletId}"
description: "Get details of a specific agentic wallet"
authMethod: "bearer"
---

<ParamField header="Authorization" type="string" required>
  Bearer token: `Bearer YOUR_ACCESS_TOKEN`
</ParamField>

<ParamField path="walletId" type="string" required>
  The wallet ID (starts with `aw_`).
</ParamField>

<RequestExample>
```bash
curl -X GET '/v1/agentic-wallets/aw_01abc123' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```
</RequestExample>
