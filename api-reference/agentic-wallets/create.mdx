---
title: "Create Agentic Wallet"
api: "POST https://crypto-sandbox.yativo.com/api/v1/agentic-wallets/create"
description: "Create a new agentic wallet for AI agent use"
authMethod: "bearer"
---

<ParamField header="Authorization" type="string" required>
  Bearer token: `Bearer YOUR_ACCESS_TOKEN`
</ParamField>

<ParamField body="name" type="string" required>
  Human-readable name for the wallet (e.g., "Production Agent Wallet").
</ParamField>

<ParamField body="chain" type="string">
  Preferred blockchain for the wallet. Defaults to the platform default.
</ParamField>

<RequestExample>
```bash
curl -X POST 'https://crypto-sandbox.yativo.com/api/v1/agentic-wallets/create' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "name": "Production Agent Wallet"
  }'
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": {
    "wallet_id": "aw_01abc123",
    "name": "Production Agent Wallet",
    "wallet_address": "0x742d35Cc6634C0532925a3b8D4C9C2A5Ef8C2B1",
    "chain": "base",
    "status": "created",
    "balance": 0,
    "created_at": "2026-04-01T12:00:00Z"
  }
}
```
</ResponseExample>
