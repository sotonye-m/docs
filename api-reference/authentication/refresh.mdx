---
title: "Refresh Token"
api: "POST https://crypto-sandbox.yativo.com/api/auth/refresh-token"
description: "Exchange a refresh token for a new access token"
authMethod: "bearer"
---

<ParamField header="Authorization" type="string" required>
  Pass the refresh token in the Authorization header: `Bearer YOUR_REFRESH_TOKEN`
</ParamField>

<RequestExample>
```bash
curl -X POST 'https://crypto-api.yativo.com/api/auth/refresh-token' \
  -H 'Authorization: Bearer YOUR_REFRESH_TOKEN'
```
</RequestExample>

<ResponseExample>
```json
{
  "status": true,
  "data": {
    "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
  }
}
```
</ResponseExample>
