---
title: "Register"
api: "POST https://crypto-sandbox.yativo.com/api/authentication/registration"
description: "Create a new user account and receive authentication tokens"
authMethod: "none"
---

<ParamField body="email" type="string" required>
  The user's email address. Must be unique.
</ParamField>

<ParamField body="password" type="string" required>
  The user's password. Must be at least 8 characters.
</ParamField>

<ParamField body="username" type="string">
  Optional display username.
</ParamField>

<ParamField body="first_name" type="string">
  The user's first name.
</ParamField>

<ParamField body="last_name" type="string">
  The user's last name.
</ParamField>

<RequestExample>
```bash
curl -X POST 'https://crypto-api.yativo.com/api/authentication/registration' \
  -H 'Content-Type: application/json' \
  -d '{
    "email": "user@example.com",
    "password": "securepassword123",
    "username": "cryptouser",
    "first_name": "Jane",
    "last_name": "Doe"
  }'
```
</RequestExample>
