---
title: "Create Card"
api: "POST https://crypto-sandbox.yativo.com/api/yativo-card/create-card"
description: "Issue a new virtual or physical card"
authMethod: "bearer"
---

<ParamField header="Authorization" type="string" required>
  Bearer token: `Bearer YOUR_ACCESS_TOKEN`
</ParamField>

<ParamField body="card_type" type="string" required>
  The type of card to create. One of: `virtual`, `physical`.
</ParamField>

<ParamField body="display_name" type="string" required>
  A label for the card shown in the dashboard, e.g. `"Business Expenses"`.
</ParamField>

<ParamField body="spending_limit_amount" type="number" required>
  The spending limit amount in USD.
</ParamField>

<ParamField body="spending_limit_frequency" type="string" required>
  How often the spending limit resets. One of: `daily`, `weekly`, `monthly`, `yearly`, `lifetime`.
</ParamField>

<RequestExample>
```bash
curl -X POST 'https://crypto-api.yativo.com/api/yativo-card/create-card' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "card_type": "virtual",
    "display_name": "Business Expenses",
    "spending_limit_amount": 2000,
    "spending_limit_frequency": "monthly"
  }'
```
</RequestExample>
