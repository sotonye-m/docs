---
title: "Get Card"
api: "GET https://crypto-sandbox.yativo.com/api/yativo-card/get/{card_id}"
description: "Retrieve full details for a specific card"
authMethod: "bearer"
---

<ParamField header="Authorization" type="string" required>
  Bearer token: `Bearer YOUR_ACCESS_TOKEN`
</ParamField>

<ParamField path="card_id" type="string" required>
  The unique ID of the card to retrieve.
</ParamField>

<RequestExample>
```bash
curl -X GET 'https://crypto-api.yativo.com/api/yativo-card/get/crd_01HX9KZMB3F7VNQP8R2WDGT4EJ' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```
</RequestExample>
