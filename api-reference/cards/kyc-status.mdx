---
title: "Get KYC Status"
api: "GET https://crypto-sandbox.yativo.com/api/yativo-card/{yativoCardId}/kyc-status"
description: "Check the KYC verification status for the card program"
authMethod: "bearer"
---

<ParamField header="Authorization" type="string" required>
  Bearer token: `Bearer YOUR_ACCESS_TOKEN`
</ParamField>

<RequestExample>
```bash
curl -X GET 'https://crypto-api.yativo.com/api/yativo-card/yativo_card_0xAbCd1234EfGh5678IjKl9012_1769031332068/kyc-status' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "message": "KYC status retrieved",
  "data": {
    "yativo_card_id": "yativo_card_0xAbCd1234EfGh5678IjKl9012_1769031332068",
    "kyc_status": "approved",
    "terms_accepted": true,
    "next_step": "create_card"
  }
}
```
</ResponseExample>
