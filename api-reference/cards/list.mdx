---
title: "List Cards"
api: "GET /yativo-card/{yativoCardId}/cards"
description: "Retrieve all cards issued under the authenticated user's card account"
authMethod: "bearer"
---

<ParamField header="Authorization" type="string" required>
  Bearer token: `Bearer YOUR_ACCESS_TOKEN`
</ParamField>

<RequestExample>
```bash
curl -X GET 'https://crypto-api.yativo.com/api/yativo-card/yativo_card_0xAbCd1234EfGh5678IjKl9012_1769031332068/cards' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```
</RequestExample>
