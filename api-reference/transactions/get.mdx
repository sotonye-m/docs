---
title: "Get Transaction"
api: "POST /v1/transactions/get-transactions"
description: "Retrieve details of a specific transaction by its ID"
authMethod: "bearer"
---

<ParamField header="Authorization" type="string" required>
  Bearer token: `Bearer YOUR_ACCESS_TOKEN`
</ParamField>

<ParamField body="transaction_id" type="string" required>
  The ID of the transaction to retrieve. Pass this as the `search` parameter to filter the results.
</ParamField>

<Note>
  This endpoint uses the same route as List Transactions. Pass the `search` field with a transaction ID or hash to retrieve a specific transaction.
</Note>

<RequestExample>
```bash
curl -X POST 'https://crypto-api.yativo.com/api/v1/transactions/get-transactions' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "search": "txn_01HX9KZMB3F7VNQP8R2WDGT4EF"
  }'
```
</RequestExample>
