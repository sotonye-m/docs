---
title: "Cancel Transaction"
api: "POST https://crypto-sandbox.yativo.com/api/transactions/get-transactions"
description: "Cancel a pending transaction before it is confirmed on-chain"
authMethod: "bearer"
---

<ParamField header="Authorization" type="string" required>
  Bearer token: `Bearer YOUR_ACCESS_TOKEN`
</ParamField>

<ParamField body="transaction_id" type="string" required>
  The ID of the pending transaction to cancel.
</ParamField>

<Note>
  Only transactions with a `pending` status can be cancelled. Once a transaction is broadcast and being confirmed on-chain, cancellation may not be possible.
</Note>

<RequestExample>
```bash
curl -X POST 'https://crypto-api.yativo.com/api/transactions/get-transactions' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "transaction_id": "txn_01HX9KZMB3F7VNQP8R2WDGT4EF"
  }'
```
</RequestExample>

<ResponseExample>
```json
{"data":[],"status":true,"message":"No transactions found"}
```
</ResponseExample>
