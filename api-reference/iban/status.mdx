---
title: "Get IBAN Status"
api: "GET /v1/iban/status"
description: "Check the status of a standalone IBAN and view its details"
authMethod: "bearer"
---

<ParamField header="Authorization" type="string" required>
  Bearer token: `Bearer YOUR_ACCESS_TOKEN`
</ParamField>

<ParamField query="customer_id" type="string">
  Filter status by customer ID. Defaults to the authenticated user if omitted.
</ParamField>

<RequestExample>
```bash
curl -X GET 'https://crypto-api.yativo.com/api/v1/iban/status' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```
</RequestExample>
