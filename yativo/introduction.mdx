Yativo is a payments infrastructure platform built for businesses, fintechs, and developers who need reliable, programmable money movement. Whether you're processing cross-border transfers, managing multi-currency accounts, or building crypto-powered financial products, Yativo gives you a single API surface to work from.

## What Yativo Does

Yativo provides two complementary platforms:

**Yativo** is the core fiat payments platform. It handles multi-currency accounts, cross-border transfers, subscription billing, and the developer tooling (API keys, webhooks, notifications) that ties it all together.

**Yativo Crypto** is the crypto infrastructure division. It extends the platform with multi-chain wallets, crypto-funded debit cards, IBAN accounts, token swaps, and a full B2B customer management layer. If you're building a crypto neobank, a treasury tool, or a Web3 payments product, Yativo Crypto is where you'll spend most of your time.

## Who It's For

- **Businesses** that need to send and receive payments across borders without building from scratch
- **Fintechs** that want to embed financial services — cards, accounts, transfers — into their products
- **Developers** who need a well-documented API with SDKs, webhooks, and sandbox environments

## Key Capabilities

| Capability | Description |
|------------|-------------|
| Multi-currency accounts | Hold and move money in multiple currencies |
| Cross-border transfers | Send funds internationally via a single API call |
| Crypto wallets | Multi-chain wallet infrastructure with automatic address generation |
| Crypto-funded cards | Issue debit cards funded from crypto balances |
| IBAN accounts | European bank account numbers linked to crypto wallets |
| Token swaps | Cross-chain and same-chain token exchange |
| Webhooks | Real-time event notifications for all platform activity |
| B2B customer management | Create and manage sub-accounts for your own customers |
